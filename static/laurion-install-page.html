FAQ

Questions?

Is this a real app or just a website?
▾
It's both — Laurion is a Progressive Web App (PWA). Once installed via Chrome, it lives on your home screen, launches fullscreen, and works offline exactly like a native Android app. The only difference is you don't need the Play Store.
Does it work on any Android phone?
▾
Yes — any Android phone with Google Chrome installed. It works on older Android versions too. For iPhone users, open Safari and tap "Share → Add to Home Screen".
How much data does it use?
▾
The initial install uses under 2 MB. After that, Laurion caches everything locally so repeat visits and offline use require zero data. Only syncing tasks and notes when reconnecting uses a tiny amount.
What can I do without internet?
▾
You can view all pages you've visited before, read cached study materials, access your planner tasks, write notes, and queue new tasks or notes for later sync. Everything you've loaded at least once is available offline.
Is Laurion free?
▾
Yes. Laurion is completely free for all students and teachers. There is no subscription, no premium tier, and no ads. It's built for real-world impact.
